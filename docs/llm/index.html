Bruk av llms.txt med LLM-er

Fri Nettleie→ LLM-bruk

llms.txt inneholder strukturert kontekst om Fri Nettleie som gjør det enklere for AI-assistenter å svare på spørsmål om nettleietariffer.

Slik bruker du det — enklest mulig

Lim inn denne lenken direkte i chatten til Claude, ChatGPT, Gemini eller en annen LLM med internettilgang:

https://kraftsystemet.no/fri-nettleie/llm/llms.txt

Legg til spørsmålet ditt, f.eks.: "Hva er tariffen for Elvia på en fredag kveld?"

For avanserte brukere

Last ned llms.txt for å bruke som systemkontekst, vedlegg eller i egne integrasjoner.

Har du spørsmål eller fant feil?

Åpne et issue på GitHub for å rapportere problemer eller komme med forslag.
